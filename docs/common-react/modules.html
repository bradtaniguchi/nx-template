HeaderBar

  • HeaderBar(props: HeaderBarProps): Element
    • Defined in lib/components/header-bar/header-bar.tsx:51

    The header-bar component provides a basic interface layout implementation for a mui based "app-bar". However, this version includes a few specific setups that are almost always used. Namely a responsive "hamburger" button, and an integrated context-provider that can be used to "provide" an internal implementation of a component, such as a "search-bar" or other UI based actions within the center of the screen.

    To correctly leverage this component, you must provide the HeaderBarContext.Provider at the "top-level" of your app, usually App.
